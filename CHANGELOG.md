# Changelog

All notable changes to VIDAS will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-12-28

### Added

#### Studio (vidas-app)
- Premium bento grid layout with glassmorphic depth system
- Custom slider, toggle, and select components with micro-interactions
- Canvas frame with CRT scanlines, vignette, and noise overlay
- Keyboard shortcuts modal (press `?`)
- Mobile-responsive slide-over drawer for controls
- Loading states and skeleton animations

#### Landing Page
- Cinematic hero with kinetic typography ("REALITY, RERENDERED")
- Animated gradient mesh background
- Bento feature grid with scroll-triggered animations
- Interactive charset showcase with horizontal scroll
- Sticky navigation with scroll-aware visibility
- Premium CTA section with pulse animations

#### Library (video2ascii)
- Comprehensive JSDoc documentation for all exports
- TypeScript strict mode compliance
- npm package.json polished for publishing
- GitHub repository configured

### Changed
- Upgraded design system to top 0.1% trend-setting aesthetic
- All native inputs replaced with custom premium components
- Enhanced accessibility with improved focus states

### Technical
- Next.js 16.1.1 + React 19.2.3
- Tailwind CSS 4 with custom design tokens
- Framer Motion for all animations
- WebGL2 for 60fps ASCII rendering
